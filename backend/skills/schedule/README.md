# 📅 ScheduleSkill (日程与待办技能)

## 📖 技能介绍
ScheduleSkill 允许 Agent 为用户管理个人时间线。它可以解析自然语言中的时间词汇，创建具体的日程、会议或待办事项提醒。

## 🎯 核心能力
- **创建日程 (`schedule:crud:create`)**: 解析用户提供的时间、地点与标题。支持处理用户所在时区 (Timezone)。
- **查询日程 (`schedule:crud:read`)**: 返回未来即将发生的日程。
- **更新日程 (`schedule:crud:update`)**: 修改已创建日程的时间或标题。
- **删除日程 (`schedule:crud:delete`)**: 批量取消或删除日程。

## ⚙️ 设计规范
- **时区处理**: 所有的时间存储在数据库中统一为 UTC，但在通过 API 返回给用户或大模型进行处理时，会根据 `user_tz` (用户时区) 进行自动转换。
- **缺失时间处理**: 若用户未指明具体时分秒（如只说了“明天”），工具内部将提供默认的时间戳（如 07:00:00）作为基准。

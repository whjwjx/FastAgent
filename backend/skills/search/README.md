# 🔍 SearchSkill (网络搜索技能)

## 📖 技能介绍
SearchSkill 赋予 Agent 连接互联网的能力。当大模型遇到需要最新资讯、实时数据（如天气、股市、新闻）或者超出了其知识截断日期的问题时，可调用此技能检索互联网信息。

## 🎯 核心能力
- **网页搜索 (`search_web`)**: 接收大模型提炼的 `query` 关键字，调用底层的搜索引擎 API 获取相关度最高的前 N 条网页结果。

## ⚙️ 设计规范
- **第三方依赖**: 默认集成 `TavilyClient` 进行深度搜索 (Advanced Depth)。
- **错误处理**: 如果未配置对应的 API Key (`TAVILY_API_KEY`)，能够进行优雅降级，向 Agent 返回友好的错误提示，而不是直接崩溃。
- **信息提炼**: 仅返回搜索结果中的标题 (`title`)、正文摘要 (`content`) 与链接 (`url`)，避免因返回完整 HTML 而导致上下文 Token 超限。
